# Example Data

Sample beneficial ownership datasets in both their original national formats and mapped to [BODS 0.4](https://standard.openownership.org/en/0.4.0/).

## BODS Examples

| File | Description |
|------|-------------|
| `mixed-ownership-bods-0.4.json` | Official BODS 0.4 example: direct & indirect ownership via an intermediary company |

## Country Mappings

Each country has an original data file and a BODS 0.4 mapped version.

### UK — Companies House PSC

| File | Format |
|------|--------|
| `uk-psc-original-data.json` | Companies House People with Significant Control (PSC) |
| `uk-psc-bods-0.4.json` | Mapped to BODS 0.4 |

### France — RNE / INPI

| File | Format |
|------|--------|
| `france-rne-original-data.json` | Registre National des Entreprises (INPI API) |
| `france-rne-bods-0.4.json` | Mapped to BODS 0.4 |

### Indonesia — AHU

| File | Format |
|------|--------|
| `indonesia-ahu-original-data.json` | Administrasi Hukum Umum (AHU) register |
| `indonesia-ahu-bods-0.4.json` | Mapped to BODS 0.4 |

### Norway — Brønnøysund Register Centre

| File | Format |
|------|--------|
| `norway-brreg-original-data.json` | Reelle Rettighetshavere (beneficial owners register) |
| `norway-brreg-bods-0.4.json` | Mapped to BODS 0.4 |
